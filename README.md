# NixOS Unpersisting

NixOS setup with unpersisting root.

## Usage

Note that this script will format the disk it points to.

1. Update the `setup.sh` variables at the top of the script

2. Run it

```bash
./setup.sh
```

3. Answer all the prompts

4. Reboot and test it

```bash
reboot
```
